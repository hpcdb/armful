DfAnalyzer framework

DfAnalyzer framework is an instantiation of ARMFUL architecture that can be coupled to existing Scientific Workflow Management Systems (SWMS) and scientific applications. DfAnalyzer keeps all the parallel execution control at the code of the SWMS or the scientific application. Besides the usage of those JAR programs by DfAnalyzer framework, another JAR program, named as Data Ingestor, has to be initialized to store provenance data, and extracted/indexed raw data from files into our relational database. More details about our provenance data model can be found here.
